# Données de test (manuel)

Ce dossier contient des fichiers simples pour tester l'application TTS sans devoir retaper du texte à chaque fois.

## Contenu
- `sample_text.txt` : un texte de démonstration à copier/coller dans l'interface.
- `api_key.example.txt` : un exemple de format pour la clé API (à renommer en `api_key.txt`).

## Utilisation rapide
1. Copiez `api_key.example.txt` vers `api_key.txt` et remplacez la valeur par votre vraie clé.
2. Lancez `python tts.py`.
3. Ouvrez `sample_text.txt` et copiez/collez le texte dans l'application.
